# react-boilerplate
### To Deploy:
* Backend
  * Run 'npm install'
  * Run 'psql < data.sql'
  * Run 'npm start'
  * This starts the backend server starting at localhost:3001
* Frontend
  * Run 'npm install'
  * Run 'npm start'
  * This starts the frontend server starting at localhost:3000

Navigate to localhost:3000/addstring to start.
